## 🚀 Getting Started

### Option 1: View Live
Visit the portfolio at: [GitHub Pages Link]

### Option 2: Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/manivarma701-jpg/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # Or simply double-click the file
   ```

3. No build process needed - it's pure HTML, CSS, and JavaScript!
